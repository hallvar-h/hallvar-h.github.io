balanceDiv

PURPOSE

SYNOPSIS

function Am1B = balanceDiv (A,B)

DESCRIPTION 

 Compute a balanced solution to A\B, handy when A is not well
 conditioned.

 To compute an inverse, enter B = eye(size(A)).

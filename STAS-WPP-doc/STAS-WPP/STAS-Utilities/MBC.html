MBC

PURPOSE

SYNOPSIS

function [TpsiB,TBpsi] = MBC (N,b1,b2,b3,psi)

DESCRIPTION 

 Define the MBC transforms for a given trio of blade DOFs.
 Dimensions not listed in b1,b2,b3 are assigned the identity.

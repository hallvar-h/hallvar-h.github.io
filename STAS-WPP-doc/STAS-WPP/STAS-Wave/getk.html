getk

PURPOSE

SYNOPSIS

function k = getk (f,d)

DESCRIPTION 

 This returns the value of k that satisfies k tanh kd = w^2/g.
 The function is the leading behavior sqrt(x) plus a polynomial
 fit through x=5, above which y=x is a very close approximation.

 Inputs:
 -------
 f               : Frequencies in Hz.
 d               : Water depth.

 Outputs:
 --------
 k               : Wave number.
